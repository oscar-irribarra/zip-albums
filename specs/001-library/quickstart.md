# Quickstart: Library management

## Prerequisites

- A local folder containing one or more album ZIP files
- The application running in development mode

## Validation scenarios

1. Start the app and confirm the library view loads albums from the configured folder.
2. Confirm each album card shows cover, name, image count, and import date.
3. Change the sort order and verify the list updates immediately.
4. Delete an album, confirm the dialog, and verify the album disappears without restarting the app.
5. Confirm that deleting one album does not remove the remaining albums.
